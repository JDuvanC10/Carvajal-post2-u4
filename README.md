# carvajal-post2-u4

## Descripción
Programa en NASM que utiliza macros, bucles y condicionales.

## Prerrequisitos
- DOSBox
- NASM
- ALINK

## Compilación
nasm -f obj programa2.asm -o programa2.obj -l programa2.lst
alink programa2.obj -o programa2.exe -entry main

## Ejecución
programa2.exe

## Evidencias
Agregar capturas en la carpeta /capturas
